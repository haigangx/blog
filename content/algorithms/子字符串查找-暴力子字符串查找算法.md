# 暴力子字符串查找算法

子字符串查找的一个最显而易见的方法就是在文本中模式可能出现匹配的任何地方检查匹配是否存在。

暴力子字符串查找的过程如下：

![暴力子字符串查找算法过程](../doc/暴力子字符串查找算法过程.jpeg)


```
//pat模式，txt文本
int search(string pat, string txt)
{
    int M = pat.size();
    int N = txt.size();
    for (int i = 0; i <= N-M; i++)
    {
        int j;
        for (j = 0; j < M; j++)
        {
            if (txt[i+j] != pat[j])
                break;
        }
        if (j == M)
            return i;   //找到匹配
    }
    return N;   //未找到匹配
}
```

在最坏的情况下(每次字符串比较都比较M次，并且每次都不匹配)，暴力子字符串查找算法在长度为N的文本中查找长度为M的模式需要`NM`次字符比较
